# わくわくはっくおせろ
特殊機能を備えたオセロゲームです
詳しくは、「わくわくはっくおせろマニュアル.pdf」に記述しています。

# 機能
* 再戦機能
* 設定変更(BGM, SE, 置ける場所表示)
* 特殊マス(革命、クロス、マス隠し、反転、マスの破壊)
* レート増減機能(自分よりレートが高い人に勝つほど増率は上がる)
* コンピュータ機能(初級、中級、上級の3レベル。上級の強さに自信あり。)
